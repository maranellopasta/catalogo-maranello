# Catálogo Maranello - Venta de productos V3

Versión ajustada para publicación en GitHub Pages.

## Cambios V3

- Galería deslizable horizontal en celular para las fotos de cada producto.
- Puntos indicadores en productos con varias fotos.
- Visor ampliado con navegación lateral, flechas y gestos táctiles.
- Soporte de teclado en escritorio: flecha izquierda, flecha derecha y Escape.

## Publicación

Sube el contenido de esta carpeta a la raíz del repositorio:

- `index.html`
- `assets/`
- `README_PUBLICACION.md`

En GitHub Pages debe estar configurado como:

- Source: Deploy from a branch
- Branch: main
- Folder: / root
